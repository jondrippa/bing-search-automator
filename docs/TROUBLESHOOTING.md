# Troubleshooting Guide

This guide covers common issues and their solutions for the Bing Rewards Automator.

## Installation Issues

### App Won't Install

**Problem**: Installation fails or gets stuck

**Solutions**:
1. Check available storage space (need at least 500MB)
2. Clear app cache: Settings → Apps → Bing Rewards → Clear Cache
3. Uninstall previous version completely
4. Restart your device
5. Try installing again

**If still failing**:
- Try using Expo Go first to test
- Check internet connection
- Restart your device completely
- Contact support with error message

### App Crashes on Launch

**Problem**: App crashes immediately after opening

**Solutions**:
1. Force close the app
2. Clear app cache and data
3. Restart your device
4. Reinstall the app
5. Update to latest version

**If still crashing**:
- Check device storage
- Update your operating system
- Try on a different device
- Contact support with crash log

## Authentication Issues

### Can't Sign In

**Problem**: Microsoft login fails or hangs

**Solutions**:

2. Verify Microsoft account credentials
3. Clear browser cache and cookies
4. Try signing in again
5. Restart the app

**If still failing**:

- Contact Microsoft support

- Keep the app open periodically
- Check account security settings
- Contact support if unauthorized access suspected

### OAuth Connection Failed

**Solutions**:
1. Verify internet connection
2. Check Microsoft account is active
3. Clear app cache
4. Restart the app
5. Try again

**If still failing**:

## Advanced Troubleshooting

### Clear App Cache (iOS)

1. Settings → General → iPhone Storage
2. Find Bing Rewards Automator
3. Tap "Offload App"
4. Tap "Reinstall App"

### Clear App Cache (Android)

1. Settings → Apps → Bing Rewards
2. Tap "Storage"
3. Tap "Clear Cache"
4. Restart the app

### Reset App Data

**Warning**: This will delete all local app data

1. Uninstall the app
2. Restart your device
3. Reinstall the app
4. Sign in again
5. Data will sync from server

### Check Logs

**iOS**:
- Connect to Mac with Xcode
- View Console output
- Check for error messages

**Android**:
- Connect to computer with Android Studio
- View Logcat
- Filter by app name
