Installation Overview

Microsoft Edge Extension
1
Download Files
Clone or download the edge-extension folder from GitHub
2
Load Extension
Open edge://extensions, enable Developer Mode, and click "Load unpacked"
3
Configure Settings
Click the extension icon and adjust tier, intervals, and anti-detection options
Android Mobile App
1
Install Expo Go
Download Expo Go from Google Play Store
2
Scan QR Code
Open Expo Go and scan the QR code from the dev server
3
Configure Settings
Tap Settings tab and adjust tier, intervals, and anti-detection options
Next Steps: After installation, proceed to the Getting Started Checklist to ensure proper setup and run your first successful search.
