Getting Started Checklist

✓
Account Setup
Verify your Bing Rewards account is active and in good standing
✓
Install Extension/App
Install Edge extension or download Android app from GitHub
✓
Configure Settings
Set tier, intervals, and enable anti-detection features
✓
Run First Search
Manually trigger one search to test the tool works
✓
Verify Points
Check Bing Rewards dashboard to confirm points awarded
✓
Check Activity Log
Review tool's activity log to see search history
✓
Create Backup Account
Set up a secondary Microsoft account as a safety measure
✓
Review Documentation
Read USER_MANUAL.md for complete feature guide
Ready to Launch: Once all items are checked, you're ready to enable automatic searches. Monitor your account weekly for any warnings or unusual activity.
